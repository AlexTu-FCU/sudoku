數獨 操作說明

遊戲規則

  • 在 9×9 的盤面填入數字 1-9。
  • 每一列、每一行、每個 3×3 宮格內，1-9 各只能出現一次。
  • 把所有空格填滿且無衝突，即為過關。

基本操作

  • 點擊選擇一格 → 該格會以深藍標示，同列/行/宮以淺藍標示。
  • 數字鍵1-9 填入數字。
  • 清除Backspace / Delete / 0 或螢幕「清」鍵清空該格。
  • 移動方向鍵 ↑ ↓ ← → 切換選擇。
  • 原始格淺灰底的格子是題目原始數字，無法修改。

顏色含義

目前選取格
同列 / 行 / 宮
相同數字
衝突格（紅）
原始題目格

筆記功能

  • 筆記 ON/OFF或按 N 切換筆記模式。開啟時按數字會在格子內標小字，不會真正填入。
  • 自動筆記掃描所有空格，自動填入每格的合法候選數字（採聯集，不覆蓋既有筆記）。
  • 清空筆記一次清掉所有筆記，但不動已填的數字。
  • 填入正式數字時，會自動清除同列/行/宮其他格的同數字筆記。

輔助功能

  • 復原或 Ctrl+Z 回到上一步（最多 200 步）。
  • 驗證檢查目前盤面有無衝突，不會洩漏答案。
  • 重置清空所有輸入回到原題目狀態（需確認）。

難度說明

  • 簡單：保留 45 格提示 — 適合新手練手感。
  • 中等：保留 36 格提示 — 標準挑戰。
  • 困難：保留 30 格提示，保證唯一解。
  • 專家：保留 26 格提示，保證唯一解，生成需 1-3 秒。

計分與計時

  • 完成獎勵：簡單 500 / 中等 1000 / 困難 2000 / 專家 3500。
  • 時間獎勵：max(0, 1000 - 秒數)，越快加越多。
  • 未完成的進度會自動存到瀏覽器，下次開啟自動繼續。

快捷鍵總覽

  • 1-9　填入數字 / 切換筆記
  • 0 Backspace Delete　清除
  • ↑ ↓ ← →　移動選擇
  • N　切換筆記模式
  • Ctrl+Z　復原
  • Esc　關閉此說明
